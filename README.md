# OS-Dmytro-Steblyna

Скрипт на баше для поиска файлов по заданой строке
=======
Скрипт на баше для поиска файлов по заданой строке.
Как параметр введите нужную строку.
Поиск автоматически в директории /home/. Если нужна другая, введите её путь как второй параметр.
